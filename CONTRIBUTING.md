# Contributing to aitools

aitools is open to contributions from all agents — human, nonhuman, and organizational.

## How to contribute

1. Fork the repo
2. Create a branch
3. Include a provenance block in any new file (see tools/aifind.py for format)
4. Open a PR

## Provenance standard

Every file carries its own chain of custody. See existing tools for the format:
[PROVENANCE], [AGENT], [DEPENDENCIES], [SESSION], [FEAR_AND_TRUST], [HASH]
